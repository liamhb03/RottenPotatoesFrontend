Welcome to Rotten Potatoes!

Looking for a way to Rickroll your friends on April 1st? Look no further than the Rotten Potatoes extension for Chrome! This extension replaces all links on every website with a Rickroll or your custom URL on April 1st. So be sure to get it today, add it to your friend's Chrome browser, and sit back.

Chrome Extension
To get the Rotten Potatoes extension, just head over to the Chrome Web Store and search for "Rotten Potatoes". Once you've found it, just click on the "Add to Chrome" button, and it will be installed. Make sure Rotten Potatoes is enabled and hidden from the tool bar. You must enable 'Allow access to file URLs' and allow site access 'On all sites'. Get it here.
GitHub
The Rotten Potatoes extension is open source and available on GitHub. If you're interested in tinkering with the code or installing it as a developmental extension, just head over to the repository and clone or download it. Download it here.

I started working on an extension for Firefox, but gave up as I couldn't solve an error. If anyone wants to modify the extension to work with different browsers and publish it here, please email me at hello@edscribe.org.

Frequently Asked Questions

Below are answers to commonly asked questions about Rotten Potatoes.

  • How can I change the URL? Change the website Rotten Potatoes redirects to by going to chrome://extensions, locating the desired extension, clicking Details, then selection the options link. Enter a valid URL and click ‘submit’. By default, it's a Rickroll. Please remain respectful.
  • How can I add Rotten Potatoes as a developmental extenstion to Chrome? https://webkul.com/blog/how-to-install-the-unpacked-extension-in-chrome/
  • Will the extension automatically run on April 1st? Yes.
  • Can I test the extension before April 1st? If you use the developmental version, you can change the date.
  • What are the permissions the extension needs? Storage. To store the URL.
  • Does Rotten Potatoes store any personal information? No.
